## 🚀 Deployment Options

### ☁️ Option A: Deploy to Render (Recommended for 24/7)
1. **Create Account**: Go to [Render.com](https://render.com) and sign up.
2. **New Service**: Click **New +** and select **Background Worker**.
3. **Connect GitHub**: Connect your GitHub account and select this repository.
4. **Environment Variables**: Go to the **Env Vars** tab and add:
   - `DISCORD_TOKEN`: Your Alt account token.
   - `LICENSE_KEY`: Your unique license key.
5. **Deploy**: Click **Create Background Worker**. Your bot is now online 24/7!

### 💻 Option B: PC Users (Windows)
1. Download the `OMT-Sentinel-v1.0.exe` from the **Releases** section.
2. Run the `.exe` and fill your details in the generated `config.json`.
3. Restart the application to access the Dashboard.
